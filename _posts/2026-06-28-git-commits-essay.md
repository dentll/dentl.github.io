# Git Commits For Future Essays

## What I Learned Today

Toay I took some notes on using secure protocols and what to do when they aren't available for the service you're using. Services like DHCP, NTP, and BGP 
don't have secure protocols that you can use. Protecting against DHCP attacks relies on detecting and then responding, rather than using some type of
secure protocol. To secure NTP you can use NTS but it is not widely adopted. The routing and switching protocol, BGP, lacks built-in security so organizations
have to design around it.
